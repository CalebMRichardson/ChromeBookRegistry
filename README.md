= Project: Centipede
A script designed to emmulate keystrokes for enrolling Chromebooks into managed environments.

== Step 1: Configuration
Change the code in the top section of the sketch file (Centipede_for_v58_and_up)

== Step 2: Load the code
Load the code onto a arduino board that works with Keyboard library

== Step 3: Prepare the Chromebook
Turn on Chromebook to be enrolled with Out-of-box-experience page. 
  By default the "Select Language" button selected  This can be changed in the Configuration

== Step 4: Execute the code
Plug in the programmed arduino to the prepared Chromebook.
